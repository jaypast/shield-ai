// LLM API costs are pass-through; all figures are operating costs, not capex
LLM cost methodology: Weighted-avg per-user API cost by tier (Free ~$0, Plus ~$2.50, Pro ~$7, Teams ~$12, Enterprise ~usage-based). Volume discounts modeled at 0% (Conservative), 15% (Moderate), 25%+ (Aggressive). Fine-tuned in-house models assumed at Aggressive scale for Tier 1 tasks.
Self-hosted infra: Modeled on AWS c5/r5 families. LiteLLM Proxy, LangGraph, Qdrant, Langfuse, and Secrets Vault are all self-hosted per the zero-retention architecture. Costs scale sub-linearly due to K8s bin-packing and reserved instance pricing.
Key variable: E2B/Firecrackers (code sandbox) is the most volatile SaaS cost — each sandbox run spins an ephemeral microVM. At scale (Aggressive), this may warrant acquiring or building a sandboxing layer in-house. Engineering headcount not included — model covers infrastructure and tooling only.
